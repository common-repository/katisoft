== Description ==

This is the KaTiSoft plugin that includes the features needed for a basic website.

If you want to contact me for any reason, feel free to email me at chithien175@gmail.com or contact me on facebook: https://www.facebook.com/ltvthienpham

== Installation ==
1. Upload the `katisoft` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
